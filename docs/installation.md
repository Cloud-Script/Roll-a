# Game Installation

Initial setup is required for the template to function correctly.

### 1. Publish the Game
Open the `.rbxl` file and **Publish to Roblox**. This step is necessary to access game settings.

### 2. Enable Services
Navigate to **Game Settings > Security** and enable:
* **Allow Studio Access to API Services**

::: danger Important!
Data saving and leaderboards will **not work** until API Services are enabled.
:::

::: warning Testing without configuration
If you want to test the template without full configuration, to allow access to we're gamepasses and products, enable:

**Allow Third-Party Sales**

You can disable it after replacing all gamepass, product, and Gift IDs with your own!
:::
